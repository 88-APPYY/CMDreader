# Novel Reader

终端命令行小说阅读器，支持 `.txt` / `.epub` 格式，单行动态阅读体验，自动记忆阅读进度，内置系统任务栏动态效果。

## 功能特性

- **单行阅读模式** — 所有文本在终端固定一行展示，逐行阅读，沉浸感强
- **动态任务栏** — 底部实时显示模拟系统任务（编译、下载、同步等），带进度条动画
- **无痕退出** — 按左右键可完全清除屏幕痕迹退出，不留下任何消息
- **记忆功能** — 自动保存/恢复阅读进度（章节 + 行号），下次继续
- **小说管理** — 支持将小说导入到项目目录中，按文件名快速打开
- **多格式支持** — `.txt` 按章节标题自动分章；`.epub` 按 HTML 结构解析
- **跨平台** — Windows / macOS / Linux 均可运行

## 安装

```bash
cd novel-reader
pip install -e .
```

依赖：`click`、`rich`、`ebooklib`（epub 支持）、`beautifulsoup4`（epub 支持）

## 命令一览

### 导入小说

将外部小说文件复制到项目的 `novels/` 目录中，方便后续按文件名直接打开：

```bash
# 导入 txt 小说
novel import ~/Downloads/斗破苍穹.txt

# 导入 epub 小说
novel import ~/Downloads/三体.epub
```

如果目标文件已存在，会提示是否覆盖。

### 查看已导入的小说

```bash
novel list
```

输出示例：

```
       已导入的小说
┌───┬────────────────────┬────────┐
│ # │ 文件名             │    大小 │
├───┼────────────────────┼────────┤
│ 1 │ 三体.epub          │ 2048 KB│
│ 2 │ 斗破苍穹.txt       │ 5120 KB│
└───┴────────────────────┴────────┘
```

### 阅读小说

```bash
# 通过文件名阅读已导入的小说（自动从 novels/ 目录查找）
novel read 斗破苍穹.txt

# 通过完整路径阅读
novel read /path/to/novel.txt

# 从指定章节开始（1-based）
novel read 斗破苍穹.txt --chapter 10

# 忽略书签，从头开始
novel read 斗破苍穹.txt --fresh
```

### 查看目录

```bash
novel toc 斗破苍穹.txt
```

### 查看阅读进度

```bash
novel bookmarks
```

显示所有已读书籍的章节和行号进度。

## 阅读界面

进入阅读后，终端分为三个区域：

```
                      第一章 初入江湖
────────────────────────────────────────────
  风吹过山岗，少年背着行囊踏上了远行的路。
────────────────────────────────────────────
════════════════════════════════════════════
[系统任务]
  🔨 编译内核模块        [█████░░░░░]  45.2% 进行中
  ⬇️ 下载更新            [████████░░]  78.5% 即将完成
  🔄 同步设置            [░░░░░░░░░░]   0.0% 正在运行
  📦 打包应用            [██████████] 100.0% 已完成
章节 1/10  行 15/200  ↑ 上一行  ↓ 下一行  ←/→ 退出
```

| 区域 | 说明                      |
| ---- | ------------------------- |
| 顶部 | 当前章节标题              |
| 中间 | 当前阅读的一行文本        |
| 底部 | 动态系统任务栏 + 操作提示 |

任务栏中的任务会自动出现、更新进度、完成并消失，模拟真实系统后台活动。

## 快捷键

| 按键           | 功能                                                   |
| -------------- | ------------------------------------------------------ |
| `↑` / `k` | 上一行                                                 |
| `↓` / `j` | 下一行                                                 |
| `←` / `h` | **无痕退出** — 清除所有屏幕痕迹，不显示任何消息 |
| `→` / `l` | **无痕退出** — 同上                             |
| `q`          | 正常退出 — 显示保存提示                               |

无痕退出会完全清除终端内容（包括滚动缓冲区），恢复终端原始状态，不留下任何阅读痕迹。

## 项目结构

```
novel-reader/
├── pyproject.toml          # 项目配置与依赖声明
├── README.md
├── novels/                 # 导入的小说存放目录（自动创建）
└── reader/
    ├── __init__.py
    ├── cli.py              # CLI 入口（import / list / read / toc / bookmarks）
    ├── display.py          # 显示引擎（单行渲染 + 任务栏 + 无痕退出）
    ├── parser.py           # 文件解析（txt 章节识别 / epub 解析）
    ├── taskbar.py          # 动态任务栏系统（8 类任务 + 进度动画）
    └── bookmark.py         # 书签持久化（~/.novel_reader_bookmarks.json）
```

## 书签存储

阅读进度保存在 `~/.novel_reader_bookmarks.json`，格式：

```json
{
  "/absolute/path/to/novel.txt": {
    "chapter": 2,
    "line": 45
  }
}
```

## 技术细节

- **Rich** 库负责终端渲染（颜色、对齐、面板）
- **Click** 框架构建 CLI 命令体系
- 后台线程独立更新任务栏状态，非阻塞键盘输入确保流畅
- 无痕退出通过 ANSI 转义序列清除屏幕 + 滚动缓冲区，并恢复终端状态
- `textwrap.wrap` 按终端宽度自动折行，确保文本不溢出
